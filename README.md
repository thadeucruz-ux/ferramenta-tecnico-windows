# 🛠 Ferramenta Técnico Windows

Ferramenta simples de manutenção para **Windows** criada em script `.bat`.

O objetivo é ajudar técnicos ou usuários a fazer limpeza e manutenção do sistema rapidamente.

---

## 🚀 Funções da ferramenta

✔ Limpeza rápida de arquivos temporários
✔ Limpeza completa do sistema
✔ Reparo de arquivos do Windows
✔ Limpeza de cache de rede
✔ Informações do computador

---

## 💻 Como usar

1. Baixe o arquivo:

```
ferramenta_tecnico.bat
```

2. Clique com botão direito.

3. Execute como **Administrador** (para funções completas).
